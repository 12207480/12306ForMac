#12306ForMac是什么
12306ForMac是Mac版的订票 /捡票 /抢票 软件。解决在Mac上订票不便的问题。
以前要么开Windows虚拟机，要么使用极其难用的官方Web，现在可以使用12306ForMac。

#功能特点
目前仅完成了基础的订票功能，后续还会继续补充功能。
代码是开源的，可以保证安全性，大家可以放心使用。
